# UTPB AR - Android Application

Senior research project done to complete graduation at UTPB. 
Can be used to:
- Locate buildings around campus
- Contact faculty
- Display a really cool 3D model of a falcon 🐦

Uses [Wikitude 💀🪦](https://www.wikitude.com/)
